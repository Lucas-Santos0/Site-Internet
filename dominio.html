O que é um dominio?

Dominio é um endereço unico para um site. geralmente ele consiste no nome do site e uma extençao de dominio. Os dominios ajudam a encontrar seu dite com facilidade sem a nesecidade de decorar seu endereço de IP

Funcionamento

Todo site tem 2 elementos principais, o dominio e o servidor de hospedagem todos os dominios são ligados a seus endereçoes de IP. quando um usuario insere um novo dominio no navegador ele procura pelo endereço de IP associado atraves da rede global de servidores DSN.Oservidor com a informaçao do endereço de IP retorna os dados ao navegador, que solicita dados sobre o site ao servidor de hospedagem do dominio, esse armazena todos os dados do site, Assim que o host tiver todos os dados de volta o navegador ira converte-los numa pagina onde os usuarios poderam visitar
